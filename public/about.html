Sobre Nosotros

API de Héroes de DC Comics.

El objetivo de este proyecto es desarrollar una API centrada en los héroes del universo de DC
Comics.La API permitirá acceder a información estructurada y detallada sobre los héroes de DC,
sus poderes, equipos y asociaciones, con el propósito de facilitar su integración en aplicaciones
web o aplicaciones móviles. Los usuarios podrán obtener datos como nombres, alias, poderes, y
afiliaciones de los héroes de DC mediante peticiones HTTP en formato JSON.

Tecnologías a Utilizar:

Para este proyecto, se seleccionaron las siguientes tecnologías:

Node.js:Plataforma de desarrollo de backend para crear el servidor.
Express:Framework de Node.js para construir la API RESTFUL.
MySQL:Base de datos relacional para almacenar y gestionar datos de héroes, poderes, y equipos.
MySQL Workbench:Herramienta de administración para crear y gestionar la estructura de la base de datos.
