Documentación

1. Funcionalidades de la API:

El proyecto incluirá las siguientes funcionalidades.

1. Consulta de héroes de DC Comics por:
Nombre y alias.
Equipos a los que pertenecen.
Poderes que poseen.

2. Consultas de equipos de DC:
Nombre del equipo.
Descripcion.

3. Consultas de poderes:
Nombre del equipo.
Descripcion.

4. Ofrecer endpoints funcionales que faciliten el acceso y filtrado de datos en la API.

2. Endpoints de la API:

La API proporcionará varios endpoints para acceder a los datos

1. Endpoints para Héroes:
GET /heroes: Obtiene todos los héroes.
GET /heroes/:id: Obtiene los detalles de un héroe específico.

2. Endpoints para Equipos:
GET /teams: Obtiene todos los equipos.
GET /teams/:id: Obtiene los detalles de un equipo específico.

3. Endpoints para Poderes:
GET /powers: Obtiene todos los poderes.
GET /powers/:id: Obtiene los detalles de un poder específico.
